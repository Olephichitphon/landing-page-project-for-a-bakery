## 🇹🇭 ภาษาไทย

โปรเจกต์หน้าเว็บ Landing Page สำหรับร้านเบเกอรี่ สร้างขึ้นเพื่อนำเสนอร้านค้า แสดงเมนูขนม และให้ข้อมูลการติดต่อ โดยเน้นการออกแบบที่สวยงาม ดึงดูด น่ารับประทาน และตอบสนองต่อผู้ใช้งาน (Responsive Design)

### 🛠 เทคโนโลยีที่ใช้ (Tech Stack)
- **React**: สำหรับสร้าง User Interface
- **Vite**: เครื่องมือสร้างโปรเจกต์ที่ทำงานได้รวดเร็ว
- **Tailwind CSS**: เฟรมเวิร์ก CSS สำหรับจัดการความสวยงามและ Layout

### 🚀 วิธีการติดตั้งและใช้งาน (Getting Started)

1. **ติดตั้ง Dependencies (แพ็กเกจที่จำเป็น)**
   เปิดเทอร์มินัลแล้วพิมพ์คำสั่ง:
   ```bash
   npm install
   ```

2. **รันโปรเจกต์ (Development Mode)**
   ```bash
   npm run dev
   ```
   จากนั้นคลิกที่ลิงก์ Local (เช่น `http://localhost:5173/`) เพื่อดูผลลัพธ์บนเบราว์เซอร์

### 📦 การสร้างไฟล์สำหรับนำไปใช้งานจริง (Production Build)
```bash
npm run build
```
ระบบจะทำการสร้างโฟลเดอร์ `dist` ซึ่งสามารถนำไปอัปโหลดขึ้น Server ได้เลย

## 🇬🇧 English

A landing page project for a bakery, built to showcase the shop, display the pastry menu, and provide contact information. The focus is on a beautiful, appetizing, and responsive design.

### 🛠 Tech Stack
- **React**: For building the User Interface
- **Vite**: A fast build tool and development server
- **Tailwind CSS**: A utility-first CSS framework for rapid UI styling

### 🚀 Getting Started

1. **Install Dependencies**
   Open your terminal and run:
   ```bash
   npm install
   ```

2. **Run the Project (Development Mode)**
   ```bash
   npm run dev
   ```
   Then click the local link (e.g., `http://localhost:5173/`) to view it in your browser.

### 📦 Production Build
```bash
npm run build
```
This will create a `dist` folder which is ready to be uploaded and deployed to your server.
